# 🌌 Infinite Library: AI-Powered Cosmic Book Recommender

Welcome to the **Infinite Library**, a premium, AI-driven book discovery platform that blends modern web aesthetics with cutting-edge Artificial Intelligence. Explore a "cosmic" collection of influential books, get personalized recommendations, and chat with an intelligent librarian.

---

## 🚀 Key Features

### 🌌 Book Discovery & Exploration
-   **Top 50 Influential Books**: A curated collection of the world's most impactful titles, displayed on the interactive homepage.
-   **Dynamic Genre Filtering (AJAX)**: Instantly switch between genres (e.g., Fiction, History, Science) with zero page reloads.
-   **Infinite Search Autocomplete**: High-performance search bar that suggests book titles from the database in real-time as you type.
-   **Detailed Book Profiles**: Dedicated pages for every book featuring metadata (Author, Publisher, Year) and high-resolution covers.

### 🧠 AI-Powered Intelligence
-   **Cosmic Librarian Chatbot**: An interactive AI assistant capable of discussing book themes, plots, and historical context.
-   **AI Book Summarization**: On-demand, premium summaries generated by **Llama 3.3-70B**, providing "Cosmic Insights" for any title.
-   **Context-Aware RAG**: The chatbot uses **Retrieval-Augmented Generation** to pull factual data from Wikipedia before answering, ensuring accuracy.
-   **Smart Session Management**: The AI remembers which book you are currently viewing, creating a personalized conversational context.

### 🎯 Recommendation Engine
-   **"More Like This" Recommendations**: Mathematical similarity-based suggestions on every book detail page.
-   **Discover Mode**: A dedicated recommendation interface where you can input a book name to find 4 highly compatible alternatives.
-   **Collaborative Filtering Logic**: Uses pre-computed similarity matrices to find patterns in user behavior and ratings.

### 🎨 Premium User Experience
-   **Interactive 3D Background**: A Three.js particle field that responds to your mouse movements.
-   **Glassmorphism Design**: A sleek, modern UI with transparent frosted-glass elements and vibrant gradients.
-   **Smooth Navigation**: Integrated with Lenis for buttery-smooth scrolling and AOS for elegant "on-scroll" entrance animations.
-   **Amazon Integration**: Quick-access links to buy any book directly from Amazon.

## 🏗️ Technical Architecture

The "Infinite Library" follows a modular architecture designed for speed, scalability, and visual excellence.

### 1. Backend: Flask & MySQL
The core logic resides in a **Flask** (Python) application, optimized for performance:
-   **MySQL Connection Pooling**: Uses `mysql.connector.pooling` to manage database connections efficiently, ensuring near-instant response times.
-   **Server-Side Caching**: Implements in-memory caching for popular genres and book lists to minimize database load.
-   **RESTful API**: Clean endpoints for book filtering, recommendations, and AI interactions.

### 2. AI & NLP (Natural Language Processing)
-   **Groq Llama 3.3 Integration**: Processes user queries and generates high-fidelity book summaries using the Groq Cloud API.
-   **Collaborative Filtering**: Uses `pickle` to load pre-calculated similarity matrices (`similarity_scores.pkl` and `pt.pkl`). It identifies patterns in user ratings to suggest books with high mathematical "compatibility."
-   **Cosine Similarity**: The mathematical core of the recommendation engine, calculating the distance between vectors in a multi-dimensional book-user space.

### 3. RAG (Retrieval-Augmented Generation)
The chatbot utilizes **RAG** to provide grounded, factual information:
-   **Wikipedia Integration**: When a user mentions a book, the system fetches a concise summary from Wikipedia.
-   **Context Injection**: This summary is injected into the LLM's prompt as "ground truth," preventing hallucinations and ensuring the AI knows exactly which book is being discussed.

### 4. Frontend & AJAX (Asynchronous JavaScript and XML)
The frontend is built for a "WOW" experience:
-   **AJAX (fetch API)**: Genre switching and chatbot messaging happen asynchronously without page reloads, providing a seamless single-page application (SPA) feel.
-   **Three.js**: Powers the interactive 3D particle field background (`scene.js`).
-   **GSAP & AOS**: Handles premium micro-animations and smooth scroll transitions.

## ⚙️ Setup & Installation

### 1. Prerequisites
-   Python 3.9+
-   MySQL Server
-   Groq API Key (Get it at [Groq Console](https://console.groq.com/))

### 2. Environment Configuration
Create a `.env` file in the root directory:
```env
GROQ_API_KEY=your_groq_api_key_here
MYSQL_HOST=localhost
MYSQL_USER=root
MYSQL_PASSWORD=your_password
MYSQL_DB=book_recommender
FLASK_SECRET_KEY=cosmic_librarian_secret
```

### 3. Installation
```powershell
# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```
